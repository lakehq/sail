# What's Next?

You have now covered what Sail is, how to use it, and how it works.
Revisit the documentation site as we add more content over time.

If you have any questions, reach out on [Slack](https://lakesail.com/slack) or [GitHub Discussions](https://github.com/lakehq/sail/discussions).

For detailed information about the Sail API and configuration options, see the [Reference](/reference/).

If you want to contribute to the project, see our [Development](/development/) handbook. We welcome [pull requests](https://github.com/lakehq/sail/pulls) on GitHub.
